# 前沿博客与资源导航

> **一句话**：当下大量最前沿的工程实践与模型进展，是各大实验室 / 大厂**先通过官方博客、再（甚至不）发论文**放出来的——读博客往往比等论文更快、更贴落地。这一页汇总我长期跟的优质源头，按"实验室博客 / 学术机构 / 推理与系统 / 个人与社区 / 资讯聚合"分类。

> **怎么用**：把它当作"信息源清单"而非阅读清单——挑 3–5 个高频更新的（如 Anthropic、OpenAI、DeepMind、Qwen、vLLM）加进 RSS 或定期翻；其余按需检索。
>
> **一点提醒**：厂商博客天然带**市场属性**，结论性数字（刷榜、加速比）要回到论文 / 技术报告核对，别直接当定论——这也是本知识库一贯的纪律。链接均为官方源；若失效以各机构官网为准。
