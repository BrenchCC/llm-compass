# 关于本站

**LLM Compass** 是一份关于 **LLM 训练算法**的中文知识图谱：从基础模型、SFT、LoRA、DPO 系列、PPO·GRPO 系列，到蒸馏、推理与解码，再到 Harness / Agent / Skills。每个算法尽量给出「一句话定义 + 核心公式 + 图解 + 调参经验」，并附原始论文与提出年份，方便快速建立全局认知、再按需深入。

收录原则：**只收讨论度高、用得最多的出名算法**——不堆砌冷门变体，也不堆砌易过期的 benchmark 数字；定性结论优先，具体数字以原论文为准。
