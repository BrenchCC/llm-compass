# About

**LLM Compass** is a knowledge compass for **LLM training algorithms** — from base models, SFT, LoRA, the DPO family and PPO·GRPO family, to distillation, inference & decoding, and Harness / Agent / Skills. Each method aims to give a one-line definition, the core formula, a diagram, and practical tuning notes, with the original paper and year of proposal.

Curation principle: **only well-known, widely-discussed algorithms** — no obscure variants, no quickly-stale benchmark numbers; qualitative conclusions first, exact figures per the original papers.
